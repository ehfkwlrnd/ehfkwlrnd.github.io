# 백준 알고리즘 13701 풀이

* 문제

  https://www.acmicpc.net/problem/13701

* 소스

  ```c++
  #include <cstdio>
  
  int A[1048576] = {0, };
  
  int main()
  {
  	int value;
  	while(~scanf("%d", &value)){
  		int* pt = A+value/32;
  		int adj = 1 << (value % 32);
  		
  		if(!(*pt & adj)){
  			*pt |= adj;
  			printf("%d ", value);
  		}
  	}
  }
  ```

* 논리

  논리는 굉장히 간단하다.

  입력을 받은 후, 과거에 입력을 받은 값이면 아무 동작도 하지 않고, 

  과거에 입력을 받지 않은 값이면 즉시 출력하면 된다.
